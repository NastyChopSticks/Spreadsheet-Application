Project Description:

The spreadsheet application is an attempt to clone what the real microsoft spreadsheet application does. It includes features like cell referencing, changing cell colors, defining functions within cells, copy and paste. The project was implemented in C# using winforms in order to replicate microsoft's real spreadsheet application as close as possible. Object oriented principles were used while developing to create a clean architecture. Principels like decoupling and high cohesion were considered during development. Some design patterns used include the command pattern, factory pattern, observer, and singleton.

Features:

Cell Referencing
Copy&Paste Cells
File Save and Load
Undo/Redo
Changing Cell Color
Function Defintion Within Cells
